# API 接口测试文档：/api_find_and_evaluate_routes

FastAPI端口：103.40.13.100:10082
## 测试输入数据

```json
{
  "start_junction": "北村",
  "end_junction": "海沧",
  "loads_ton": [10, 15, 12],
  "spacings": [3.5, 1.2],
  "max_path_length": null,
  "top_n": 3
}
```

### 输入参数说明

- `start_junction`: 起点枢纽点名称，设置为"北村"
- `end_junction`: 终点枢纽点名称，设置为"海沧"
- `loads_ton`: 轴重吨数列表，表示车辆的轴重配置 [10吨, 15吨, 12吨]
- `spacings`: 轴距列表，表示车辆各轴之间的距离 [3.5米, 1.2米]
- `max_path_length`: 最大路径长度限制，设置为null表示不限制
- `top_n`: 返回的最佳路线数量，设置为3表示返回前3条最佳路线
