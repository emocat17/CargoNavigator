### 表单字段来源说明

| 字段类别 | 来源 | 说明 |
|---------|------|------|
| 申请编号 | 系统自动生成 | 唯一标识符 |
| 货物名称 | 第二步：关键参数 | 用户输入 |
| 通行开始/结束时间 | 第二步：关键参数 | 用户输入 |
| 起运省/沿线省 | 系统自动生成 | 根据出发地和路线自动填充 |
| 出发地/目的地 | 第二步：关键参数 | 用户输入 |
| 通行路线 | 第三步：路线选择 | 系统计算，用户选择 |
| 货物质量 | 第二步：关键参数 | 用户输入 |
| 车货总质量 | 系统自动计算 | 货物质量+牵引车整备质量+挂车整备质量 |
| 轴数/轴距/轴荷分布 | 第二步：关键参数 | 用户输入，用于桥梁效应计算 |
| 轮胎数 | 第四步：补充信息 | 用户输入 |
| 货物外廓尺寸 | 第四步：补充信息 | 用户输入 |
| 车货总体外廓尺寸 | 第二步：关键参数 | 用户输入 |
| 车货总体轮廓图 | 第四步：补充信息 | 用户上传 |
| 业户信息 | 第一步：模板信息 | 用户输入，可保存为模板 |
| 经办人信息 | 第一步：模板信息 | 用户输入，可保存为模板 |
| 牵引车信息 | 第一步：模板信息 | 用户输入，可保存为模板 |
| 挂车信息 | 第一步：模板信息 | 用户输入，可保存为模板 |
| 护送方案文件 | 第四步：补充信息 | 用户上传 |
