## 定时任务设置

### 使用Crontab设置定时任务

#### 1. 打开Crontab编辑界面
```bash
crontab -e
```

#### 2. 添加定时任务
```bash
# 每30分钟运行一次爬虫脚本
*/30 * * * * /usr/bin/python3 /home/Spider/main.py

# 每30分钟运行一次自定义脚本
*/30 * * * * /usr/bin/python3 /path/to/your/main.py
```

## 环境依赖

### Chrome浏览器安装（Linux系统）

如果系统尚未安装Chrome，需要先安装：

1. **更新软件包列表**：
    ```bash
    sudo apt update
    ```

2. **安装Chrome**：
    ```bash
    wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
    sudo apt install ./google-chrome-stable_current_amd64.deb
    ```

3. **验证安装**：
    ```bash
    which google-chrome
    ```

4. **安装playwright浏览器驱动**：
    ```bash
    playwright install
    ```

## 常见问题解决

### DrissionPage找不到Chrome浏览器

错误信息：`FileNotFoundError: [Errno 2] No such file or directory: 'chrome'`

**解决方案**：
1. 确保Chrome浏览器已正确安装
2. 检查Chrome是否在系统PATH中
3. 安装playwright浏览器驱动

### 爬虫脚本定时运行

确保：
1. Python路径正确（使用`which python3`确认）
2. 脚本文件具有执行权限
3. 环境变量设置正确
4. 依赖库已安装
