## 1.py 脚本处理流程

`1.py`脚本用于处理单个桥梁的影响线数据文件，计算最大效应值。处理流程如下：

### 1. 定义冲击系数计算函数
- 代码先定义了一个`get_damping(f)`函数，用于根据结构基频计算冲击系数
- 计算规则：
  - 当频率f < 1.5Hz时，冲击系数u = 0.05
  - 当1.5Hz ≤ f ≤ 14Hz时，冲击系数u = 0.1767 * ln(f) - 0.0157
  - 当f > 14Hz时，冲击系数u = 0.45

### 2. 读取影响线数据文件
- 代码读取指定的影响线数据文件（如：K0+15九龙江大桥-负弯矩-大件运输车道.txt）
- 使用`pd.read_csv`函数，以空格为分隔符，gbk编码读取文件

### 3. 数据预处理
代码对原始影响线数据进行以下处理步骤：
1. **去掉前3行**：使用`data.iloc[3:, :]`删除文件前3行的标题信息
2. **去掉最后2行**：使用`data.iloc[:-2, :]`删除文件末尾的统计信息
3. **保留前四列**：使用`data.iloc[:, :4]`只保留前4列数据（单元号、位置、距离、影响值）
4. **将第一行作为表头**：将原第4行作为表头，并重置索引
5. **删除第二列**：删除位置列（第2列），保留单元号、距离和影响值列
6. **去掉第2列中相邻重复的行**：删除距离列中相邻重复的行，确保数据点不重复

### 4. 数据分割与合并
代码对预处理后的数据进行以下操作：
1. **按行数均分为两部分**：将数据均分为前半部分和后半部分
2. **将后半部分最后一列赋值到前半部分新列"INFLUENCE2"**：将后半部分的影响值列赋值给前半部分的新列
3. **计算第3列与第4列的平均值，作为新列"INFLUENCE_combined"**：计算原影响值列和新影响值列的平均值，得到综合影响值

### 5. 保存处理后的数据
- 将处理后的数据保存为与原文件同名的Excel文件（如：K0+15九龙江大桥-负弯矩-大件运输车道.xlsx）

### 6. 定义车辆参数
- 定义车辆轴重：`loads_ton = [10, 10, 10] + [17.98] * 5`（单位：吨）
- 转换为千牛：`loads = [p * 9.8 for p in loads_ton]`（单位：kN）
- 定义车辆轴距：`spacings = [3.2, 1.4, 7] + [1.45] * 4`（单位：米）
- 计算轴位置：`axle_positions = np.cumsum([0] + spacings)`（单位：米）

### 7. 移动计算最大效应
代码通过以下步骤计算车辆在桥梁上移动时的最大效应：
1. **提取横坐标和影响线数据**：从处理后的数据中提取距离列和综合影响值列
2. **创建插值函数**：使用`scipy.interpolate.interp1d`创建线性插值函数，用于计算任意位置的影响值
3. **确定桥梁范围**：`bridge_start = min(x)`和`bridge_end = max(x)`确定桥梁的起始和结束位置
4. **移动计算**：
   - 设置移动步长：`step = 0.01`（米）
   - 从车头进桥前开始，到最后一轴出桥结束，以步长移动车辆
   - 对于每个位置，计算所有轴在桥梁上产生的影响值之和
   - 记录最大效应值（保留符号）

### 8. 应用冲击系数
- 将计算得到的最大效应值乘以冲击系数和安全系数：`max_effect_with_impact = max_effect * (1 + u) * 1.1`
- 输出最终结果

## 各桥大件运输下的效应.py 脚本处理流程

`各桥大件运输下的效应.py`脚本用于批量处理所有桥梁的影响线数据，计算各桥梁的最大效应值。处理流程如下：

### 1. 定义车辆参数
- 与`1.py`脚本相同，定义车辆轴重、轴距和轴位置

### 2. 读取桥梁基本信息
- 读取"桥梁表.xlsx"文件，获取所有桥梁的基本信息
- 使用`pd.read_excel`函数读取Excel文件

### 3. 桩号转换与排序
- 定义`parse_station`函数，将桩号字符串（如"K0+15"）转换为数值（如15）
- 为每座桥梁计算桩号数值，并按桩号数值排序

### 4. 定义冲击系数计算函数
- 与`1.py`脚本相同，定义`get_damping`函数用于计算冲击系数

### 5. 批量处理桥梁数据
代码遍历每座桥梁，执行以下操作：

#### 5.1 匹配桥梁文件夹
- 根据桩号匹配对应的桥梁文件夹（允许桩号后有文字，大小写不敏感）
- 如果找不到对应文件夹，记录错误信息并跳过该桥梁

#### 5.2 计算冲击系数
- 从桥梁基本信息中获取结构基频
- 调用`get_damping`函数计算冲击系数

#### 5.3 查找影响线数据文件
- 在桥梁文件夹中查找所有txt文件
- 根据文件名识别正弯矩、负弯矩和剪力的影响线数据文件

#### 5.4 读取并处理影响线数据
定义`read_txt`函数，用于读取和处理影响线数据文件，该函数执行以下操作：

1. **读取文件**：使用`pd.read_csv`函数读取txt文件
2. **数据预处理**：与`1.py`脚本相同的预处理步骤
3. **数据分割与合并**：与`1.py`脚本相同的数据分割与合并步骤
4. **保存处理后的数据**：将处理后的数据保存为Excel文件
5. **提取横坐标和影响线数据**：从处理后的数据中提取距离列和综合影响值列
6. **创建插值函数**：使用`scipy.interpolate.interp1d`创建线性插值函数
7. **确定桥梁范围**：确定桥梁的起始和结束位置

#### 5.5 移动计算最大效应
代码通过以下步骤计算车辆在桥梁上移动时的最大效应：

1. **正向移动计算**：
   - 从车头进桥前开始，到最后一轴出桥结束，以步长移动车辆
   - 对于每个位置，计算所有轴在桥梁上产生的影响值之和
   - 记录最大效应值

2. **反向移动计算**：
   - 从车尾出桥后开始，到第一轴进桥结束，以步长反向移动车辆
   - 对于每个位置，计算所有轴在桥梁上产生的影响值之和
   - 记录最大效应值

3. **取绝对值最大的效应**：比较正向和反向计算得到的最大效应值，取绝对值较大的一个

4. **应用冲击系数**：将计算得到的最大效应值乘以冲击系数和安全系数

#### 5.6 收集结果
将每座桥梁的计算结果（桩号、正弯矩最大效应、负弯矩最大效应、剪力最大效应、冲击系数）收集到列表中

### 6. 保存结果
- 将收集到的结果转换为DataFrame
- 保存为"桩号最大效应.xlsx"文件

## 关键技术点

### 1. 影响线数据处理
- 原始影响线数据包含大量冗余信息，需要通过预处理提取有效数据
- 通过分割和合并数据，得到更准确的影响线值
- 删除重复数据点，提高计算效率

### 2. 车辆移动模拟
- 使用插值函数计算任意位置的影响值
- 通过小步长移动车辆，精确计算最大效应值
- 同时考虑正向和反向移动，确保不遗漏任何可能的最大效应位置

### 3. 冲击系数计算
- 根据结构基频计算冲击系数，考虑不同频率范围的计算公式
- 将冲击系数应用于最大效应值，得到更真实的结果

### 4. 批量处理
- 自动识别桥梁文件夹和影响线数据文件
- 统一处理所有桥梁数据，提高效率
- 自动保存处理结果，便于后续分析

## 总结

这两个Python脚本实现了从原始影响线数据到最大效应值的完整处理流程。`1.py`脚本专注于单个桥梁的处理，而`各桥大件运输下的效应.py`脚本则实现了批量处理功能。通过这些脚本，可以高效地分析大件运输车辆对桥梁的效应，为桥梁安全评估提供依据。

处理流程主要包括：数据预处理、数据分割与合并、车辆移动模拟、最大效应计算和冲击系数应用等步骤。最终结果保存在"桩号最大效应.xlsx"文件中，包含所有桥梁的最大效应值和冲击系数。
