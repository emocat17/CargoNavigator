## 快速开始

1. **服务器部署**: 首先阅读 [01-服务器配置与部署.md](./01-服务器配置与部署.md) 了解系统部署要求
2. **API集成**: 查看 [02-Dify_API使用指南.md](./02-Dify_API使用指南.md) 集成Dify API服务
3. **爬虫使用**: 参考 [03-爬虫系统使用指南.md](./03-爬虫系统使用指南.md) 配置爬虫系统
4. **知识库配置**: 按照 [04-Dify知识库与配置指南.md](./04-Dify知识库与配置指南.md) 配置知识库
5. **技术规范**: 阅读 [05-公路大件运输技术规范.md](./05-公路大件运输技术规范.md) 了解技术标准
6. **申请流程**: 参考 [06-运输申请表单指南.md](./06-运输申请表单指南.md) 了解申请流程
7. **计算说明**: 查看 [07-桥梁效应比值计算说明.md](./07-桥梁效应比值计算说明.md) 了解桥梁评估方法

## 常见问题快速解决

### 连接问题
- 检查服务器配置和网络连接
- 确认API端点和密钥正确性

### 环境问题
- 安装必要的依赖库
- 配置Chrome浏览器

### 数据问题
- 检查爬虫脚本运行状态
- 确认文件上传机制正常工作
