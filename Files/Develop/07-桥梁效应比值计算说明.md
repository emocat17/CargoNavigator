# 桥梁效应比值计算说明

## 概述
本文档详细说明了桥梁效应比值计算系统所需的参数和计算方法。这些参数主要用于评估大件运输车辆对桥梁的影响，确保运输安全。

## 1. 车辆参数（从司机处收集）

### 1.1 车辆基本信息
- **车辆总重** (单位: 吨)
  - 描述: 车辆的总重量，包括货物和车辆自重
  - 示例: 100吨

### 1.2 轴重信息
- **轴重分布** (单位: 吨)
  - 描述: 车辆各个轴的重量分布
  - 示例: [10, 10, 10, 17.98, 17.98, 17.98, 17.98, 17.98]
  - 说明: 当前代码中使用了8个轴，前三个轴各10吨，后五个轴各17.98吨

### 1.3 轴距信息
- **轴距分布** (单位: 米)
  - 描述: 车辆相邻轴之间的距离
  - 示例: [3.2, 1.4, 7, 1.45, 1.45, 1.45, 1.45]
  - 说明: 当前代码中使用了7个轴距值

### 1.4 其他车辆参数
- **轴数**
  - 描述: 车辆的轴数
  - 示例: 6
  
- **轮胎数**
  - 描述: 车辆的轮胎数
  - 示例: 22
  
- **货物质量** 
  - 描述: 货物的质量
  - 示例: 50吨

- **车货总质量** 
  - 描述: 车辆和货物的总质量
  - 示例: 150吨
  - 说明: 可通过货物质量+牵引车整备质量+挂车整备质量自动计算

- **货物外廓长宽高** (单位: 米)
  - 描述: 货物的长度、宽度和高度
  - 示例: 20m x 4.9m x 3.9m

- **车货总体外廓长宽高** (单位: 米)
  - 描述: 车辆和货物的总体外廓尺寸
  - 示例: 24m x 4.9m x 4.7m

## 2. 桥梁参数（系统预设或从数据库获取）

### 2.1 桥梁基本信息
- **桩号** 
  - 描述: 桥梁在路线上的位置标识
  - 示例: "K0+15", "K0+9", "K0+3"

- **桥型**
  - 描述: 桥梁的结构类型
  - 示例: "变截面连续梁桥", "T梁", "空心板"

- **跨径** (单位: 米)
  - 描述: 桥梁的跨度
  - 示例: "40+70+40", "3*40", "16"

### 2.2 桥梁结构参数
- **梁片数**
  - 描述: 桥梁横截面的梁片数量
  - 示例: 1, 9

- **车道数**
  - 描述: 桥梁的车道数量
  - 示例: 3

- **大件车行驶车道**
  - 描述: 大件车行驶的车道编号
  - 示例: 2

- **控制截面**
  - 描述: 桥梁的控制截面位置
  - 示例: 2

### 2.3 桥梁动力特性
- **结构基频** (单位: Hz)
  - 描述: 桥梁结构的基频，用于计算冲击系数
  - 示例: 2.39, 2.689, 6.16

### 2.4 桥梁影响线数据
- **正弯矩影响线数据**
  - 描述: 桥梁正弯矩影响线数据，用于计算车辆荷载引起的正弯矩效应
  - 数据来源: 桥梁分析软件(如MIDAS)导出的txt文件

- **负弯矩影响线数据**
  - 描述: 桥梁负弯矩影响线数据，用于计算车辆荷载引起的负弯矩效应
  - 数据来源: 桥梁分析软件(如MIDAS)导出的txt文件

- **剪力影响线数据**
  - 描述: 桥梁剪力影响线数据，用于计算车辆荷载引起的剪力效应
  - 数据来源: 桥梁分析软件(如MIDAS)导出的txt文件

## 3. 计算过程

### 3.1 冲击系数计算
根据桥梁结构基频计算冲击系数，公式如下：
```
如果 f < 1.5: u = 0.05
如果 1.5 ≤ f ≤ 14: u = 0.1767 * ln(f) - 0.0157
如果 f > 14: u = 0.45
```

### 3.2 影响线数据处理
1. 从txt文件读取影响线数据
2. 去除前3行和后2行的无关数据
3. 保留前4列数据
4. 将第一行作为表头
5. 删除第二列
6. 去除第二列中相邻重复的行
7. 将数据分为两部分
8. 计算影响线平均值

### 3.3 车辆荷载效应计算
1. 将车辆轴重转换为kN单位
2. 计算轴位置累积值
3. 使用线性插值方法构建影响线函数
4. 模拟车辆在桥上移动，计算最大效应
5. 考虑正向和反向行驶两种情况
6. 乘以冲击系数(1+u)和1.1的安全系数

### 3.4 效应比值计算
效应比值 = 车辆荷载效应 / 桥梁设计值
- 正弯矩效应比值 = 正弯矩最大效应 / 正弯矩设计值
- 负弯矩效应比值 = 负弯矩最大效应 / 负弯矩设计值
- 剪力效应比值 = 剪力最大效应 / 剪力设计值

## 4. 结果评估
- 如果效应比值 < 1，表示车辆可以安全通过桥梁
- 如果效应比值 ≥ 1，表示车辆可能对桥梁造成损害，需要采取额外措施

## 5. 注意事项
1. 车辆参数必须准确，特别是轴重和轴距分布
2. 桥梁影响线数据必须准确反映桥梁的实际受力特性
3. 冲击系数计算考虑了桥梁的动力特性，对结果影响较大
4. 安全系数1.1是额外考虑的安全储备
