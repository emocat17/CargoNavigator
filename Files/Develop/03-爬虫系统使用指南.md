# 爬虫系统使用指南

## 概述

本爬虫系统用于抓取高速路况和施工信息，并将数据保存为Markdown格式文件。

## 数据源

- https://www.fjetc.com/traffic-info
<!-- - https://qz.bendibao.com/mtools/gaosu/ -->

## 使用流程

### 1. 提取事件代码和URL

首先运行`extract_event_details.py`，提取事件代码和URL。

```bash
python extract_event_details.py
```

**可选步骤**：手动检查提取结果，可测试单个文件提取`extract_event_details.py`，运行确认无误后，继续下一步。

### 2. 批量提取事件详情

运行`batch_extract_event_details.py`，批量提取事件详情。

```bash
python batch_extract_event_details.py
```

## 技术依赖

### Chrome浏览器

爬虫系统使用DrissionPage库，需要Chrome浏览器支持。

#### Linux系统安装Chrome

如果系统尚未安装Chrome，需要先安装：

1. **更新软件包列表**：
    ```bash
    sudo apt update
    ```

2. **安装Chrome**：
    ```bash
    wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
    sudo apt install ./google-chrome-stable_current_amd64.deb
    ```

3. **验证安装**：
    ```bash
    which google-chrome
    ```

4. **安装playwright浏览器驱动**：
    ```bash
    playwright install
    ```

## 常见问题解决

### DrissionPage找不到Chrome浏览器

**错误信息**：
```
FileNotFoundError: [Errno 2] No such file or directory: 'chrome'
```

**问题原因**：
DrissionPage尝试自动启动名为`chrome`的可执行文件，但在系统的环境变量`PATH`所包含的目录里没有找到它。

**解决方案**：
1. 确保Chrome浏览器已正确安装
2. 检查Chrome是否在系统PATH中
3. 安装playwright浏览器驱动

## 数据上传

爬虫系统支持将抓取的数据上传到Dify平台：

1. 运行主脚本：
   ```bash
   python main.py
   ```

2. 系统会自动：
   - 抓取最新数据
   - 将数据保存为Markdown文件
   - 上传新文件到Dify平台

3. 文件上传机制：
   - 系统维护已上传文件列表（存储在`.uploaded_files.json`）
   - 仅上传新文件，避免重复上传
   - 使用文件名作为唯一标识

## 定时运行

可以使用Crontab设置定时任务，定期运行爬虫：

```bash
# 每30分钟运行一次
*/30 * * * * /usr/bin/python3 /path/to/main.py
```

## 注意事项

1. 确保网络连接正常，可以访问目标网站
2. 定期检查爬虫脚本是否正常运行
3. 监控上传文件大小和数量，避免存储空间不足
4. 如遇到反爬机制，可能需要调整爬取频率或添加代理
